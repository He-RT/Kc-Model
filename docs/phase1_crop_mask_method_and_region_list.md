## 2. 从本地文献提取到的可复用方法

本次优先检查了 `/Users/hert/Projects/dachuang` 中的本地文献。与“冬小麦种植区提取”最直接相关的是：

从该论文提取到的方法要点如下：

### 2.1 论文中的原始做法

- 研究区：河北省邯郸市邱县
- 数据源：`Sentinel-2 L2A`
- 时段：`2022-2023`
- 方法：监督分类中的最大似然法
- 目标：土地利用分类后重点提取冬小麦种植区
- 样本：结合地面调查样点
- 精度评价：总体精度与 `kappa`

从论文正文可提取到的关键信息：

- 论文明确写到“基于 2022-2023 年 Sentinel-2 L2A 卫星影像数据，通过最大似然法对研究区域的土地利用情况进行分类并着重提取了冬小麦种植区的空间分布信息”
- 论文给出的分类效果为：
  - 总体精度 `98.94%`
  - `kappa = 0.98`

### 2.2 这条方法为什么能作为“第一部分”的起点

对你当前的项目目标来说，这篇论文有两个直接价值：

1. 它证明了 `Sentinel-2` 做冬小麦识别在河北平原场景里是可行的。
2. 它给了一个最容易跑通的首版路线：
   `Sentinel-2 L2A -> 监督分类 -> 冬小麦掩膜`

### 2.3 但不能原样照搬的地方

这篇论文的方法适合“县域、小范围、单年/双年、样本人工控制较强”的场景，不适合直接扩展成“华北平原 2021-2025 多年统一生产线”，原因有三点：

- 最大似然法对训练样本质量和单期光谱可分性依赖较强
- 县域方法通常默认地类背景比较稳定，跨省跨年时鲁棒性不足
- 论文更偏“局地提取 + 后续 Kc 估算”，不是面向区域多年自动化生产线设计的

## 3. 面向本项目的稳妥改造建议

基于上面那篇论文的方法，但为了适配“华北平原多年单作物”任务，第一部分建议采用下面这条更稳的工程路线。

### 3.1 推荐识别框架

```text
耕地掩膜
  -> 冬小麦关键物候窗口筛选
  -> Sentinel-2 多时相指数特征
  -> 监督分类/规则筛选
  -> 冬小麦年度掩膜
```

### 3.2 MVP 识别方法建议

先做可跑通版本时，推荐顺序如下：

1. `Sentinel-2 L2A` 作为主数据源
2. 先使用冬小麦关键生育期的多时相特征，而不是只用单景
3. 分类器可以先保留为“最容易解释”的监督分类方案
4. 如果首版最大似然法不稳，直接切到 `Random Forest`

### 3.3 对 GEE 最友好的首版方案

结合 GEE 使用便利性，首版我建议这样做：

- 输入：
  - `Sentinel-2 SR Harmonized`
  - 耕地掩膜
  - 华北平原行政区边界
- 特征：
  - `NDVI`
  - `EVI`
  - `LSWI/NDWI`
  - 红边指数如 `NDRE`
  - 多时相窗口统计量
- 分类：
  - 优先 `Random Forest`
- 输出：
  - `yearly winter wheat mask`

原因很直接：

- 比最大似然法更适合大区域自动化
- 在 GEE 里实现简单
- 后续扩到夏玉米也方便

## 4. 近五年华北平原冬小麦种植区域列表

### 4.1 口径说明

这里的“华北平原冬小麦种植区域列表”不是全国所有冬小麦主产省，而是为你的项目服务的“华北平原研究范围清单”。

我把它分成两层：

- `核心区域`：建议直接纳入首版 GEE 批处理
- `扩展区域`：地理上属于华北平原边缘或冬小麦面积较小，可第二步再纳入

### 4.2 外部核验依据

为避免只凭经验列省份，我额外核验了近年的公开资料：

- 中国农科院 `2023-11-27` 发布的《2023-2024年度全国冬小麦秋冬种生产形势研判》
  其中明确提到冬小麦主产省包括：
  `河南、山东、江苏、安徽、河北` 等
- 农业农村部 `2025-04-20` 发布的《农产品供需形势分析月报》
  其中在冬小麦主产省苗情描述里列出：
  `河南、山东、河北、安徽、江苏` 等

基于这两条较新的公开信息，并结合“华北平原”空间范围，本项目第一阶段采用以下区域口径。

### 4.3 核心区域列表

`2021-2025` 五个年份统一采用以下核心区域：

- 河北省平原农区
- 河南省黄淮海冬麦区
- 山东省平原农区
- 安徽省沿淮及淮河以北地区
- 江苏省淮河及苏北灌溉总渠以北地区

### 4.4 扩展区域列表

可作为第二批纳入：

- 北京市平原耕地区
- 天津市耕地集中区

### 4.5 不建议在第一批纳入的区域

虽然这些省份也属于冬小麦主产省或主产带的一部分，但不建议在“华北平原首版”里先纳入：

- 山西
- 陕西
- 湖北
- 甘肃
- 新疆

原因：

- 它们要么不属于华北平原主体
- 要么生态区差异明显
- 要么会把第一批 AOI 过早做大，影响作物掩膜稳定性
